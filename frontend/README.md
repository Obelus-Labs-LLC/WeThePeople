# WeThePeople Frontend (minimal scaffold)

This is an unstyled Vite + React + TypeScript scaffold intended to unblock UI work.

## Configure

Backend base URL is read from `VITE_API_BASE_URL` (defaults to `http://localhost:8000`).

Example (PowerShell):

```powershell
$env:VITE_API_BASE_URL = "http://localhost:8000"
```

## Run

```powershell
cd frontend
npm install
npm run dev
```

Then open the URL printed by Vite (usually `http://localhost:5173`).
