# The Influence Journal

Data-driven investigations into corporate influence on government. Part of the [WeThePeople](https://wethepeopleforus.com) ecosystem.

## Planned Features

Pages and features that will migrate here from the main WeThePeople frontend:

- **Stories** — Data-driven narratives about corporate influence (from StoriesPage)
- **Story Detail** — Full story view with embedded data visualizations (from StoryDetailPage)
- **Blog** — Editorial posts and analysis
- **Data Investigations** — Long-form deep-dives with interactive charts
- **Newsletter** — Weekly digest signup and archive

## Development

```bash
cd sites/journal
npm install
npm run dev    # http://localhost:5175
```

## Stack

React 19 + Vite + Tailwind CSS 4 + TypeScript
