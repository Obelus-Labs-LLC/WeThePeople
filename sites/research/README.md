# WTP Research

Deep-dive research tools for civic data. Part of the [WeThePeople](https://wethepeopleforus.com) ecosystem.

## Planned Features

Pages that will migrate here from the main WeThePeople frontend:

- **Patent Explorer** — Search USPTO patents by company, technology area, or keyword
- **Drug Lookup** — FDA approvals, clinical trial pipelines, CMS payment data
- **Clinical Trial Tracker** — Browse and filter active clinical trials by company and phase
- **Insider Trades Dashboard** — SEC Form 4 filings with filtering and trend analysis
- **Company Financials** — SEC EDGAR filings, stock data, financial metrics
- **Macro Indicators** — FRED economic data, Fed press releases, market movers

## Development

```bash
cd sites/research
npm install
npm run dev
```

## Stack

React 19 + Vite + Tailwind CSS 4 + TypeScript
